1) Image Basic src/alt

รูปภาพใน HTML ใช้แท็ก img โดยต้องมี src และควรมี alt เสมอ

<img src="images/photo.jpg" alt="ภาพตัวอย่าง">

alt สำคัญมาก: ช่วย SEO + ใช้แทนภาพเมื่อโหลดไม่ได้ + ช่วยผู้ใช้ Screen Reader

2) Resize & Aspect Ratio

ถ้ากำหนดทั้งความกว้างและสูง ให้ใช้ object-fit เพื่อไม่ให้ภาพบิด

/* ครอปภาพให้พอดีกรอบ */
.img-fixed {
  width: 260px;
  height: 160px;
  object-fit: cover;
}

cover = ครอปให้เต็มกรอบ, contain = เห็นภาพครบ (อาจมีขอบว่าง)

3) Object fit contain fill cover

ตัวอย่างการใช้ object-fit แบบต่างๆ

ภาพต้นฉบับ

contain: เห็นภาพครบ (มีขอบว่าง)

fill: ยืดเต็มกล่อง (ภาพเบี้ยว)

cover: เต็มกล่องสวยงาม (ตัดส่วนเกิน)

4) Image Link a+img

คลิกที่รูปเพื่อเปิดลิงก์ เหมาะกับ banner / thumbnail

<a href="https://www.youtube.com" target="_blank">
  <img src="images/banner.jpg" alt="Banner">
</a>

ใช้ target="_blank" เมื่ออยากเปิดในแท็บใหม่

5) Gallery & Upload JS + Remove

แสดงรูป Grid เพิ่มรูปจากเครื่องได้ และกดปุ่ม X เพื่อลบรูป

💡 ใช้ position: relative ที่กล่องแม่ และ absolute ที่ปุ่ม เพื่อจัดตำแหน่งมุมขวาบน

6) Responsive Images max-width

ทำให้ภาพไม่ล้นการ์ดและย่อขยายตามหน้าจอ

img {
  max-width: 100%;
  height: auto;
}

7) Figure + Caption semantic

ใช้แท็ก <figure> และ <figcaption> เพื่ออธิบายภาพ

8) ใช้รูปจากเครื่อง Local

จัดโฟลเดอร์รูปให้เป็นระเบียบ เช่น images/ แล้วอ้างอิง path

my_project/
  ├─ index.html
  └─ images/
      └─ image.png

<img src="images/image.png" alt="image card 8">
