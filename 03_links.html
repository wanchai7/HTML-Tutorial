Workshop: Various Links

เรียนรู้การสร้างลิงก์รูปแบบต่างๆ พร้อมตัวอย่างและแบบฝึกหัด

1. External Links

การลิงก์ไปยังเว็บไซต์ภายนอก

🔗 Visit Wikipedia
target="_blank" (เปิดแท็บใหม่)

🔗 Visit Google
target="_self" (เปิดหน้าเดิม)

2. Internal & Anchor Links

การลิงก์ภายในหน้าเว็บเดียวกัน (Jump)

<a href="#jump">Jump to Section</a>
...
<h3 id="jump">Target</h3>
Click to Jump ⬇️

🎯 This is the jump target.

3. Email Link

เปิดโปรแกรมส่งอีเมลอัตโนมัติ

<a href="mailto:email@ex.com">
  Send Email
</a>
📧 Send an Email

4. Download Links

บังคับให้ดาวน์โหลดไฟล์แทนการเปิดดู

<a href="file.pdf" download>Download</a>
⬇️ Download File
ต้องมีไฟล์จริงในโฟลเดอร์ถึงจะโหลดได้

5. Button Links & Styles

การใช้ CSS ตกแต่ง <a> หรือ <button> ให้สวยงาม

<button onclick="location.href='url'">Link</button>

6. Image Link

ใช้รูปภาพเป็นลิงก์ (คลิกที่รูป YouTube ด้านล่าง)

<a href="https://youtube.com">
  <img src="image.jpg">
</a>
หากต้องการใช้รูปของตนเอง ให้เก็บไว้ในโฟลเดอร์ images/photo.png
